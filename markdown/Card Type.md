# Card Type
There are six different card types in _Sorcery_:

- Avatar
- Minion
- Artifact
- Aura
- Magic
- Site

## Multiple Simultaneous Card Types
It is sometimes possible for cards to have multiple types. For example, automatons are minions and artifacts.

For more details, see characteristics and "Cards with Multiple Types."
